# NeuronaGenericaConFuncionDeActivacion


Para este ejemplo hemos usado una clase llamada Neuron.

Aplicación con Python y Streamlit "Neurona genérica" con las siguientes características:

+ Vector de entrada
+ Vector de pesos
+ Sesgo
+ Función de activación
